Paxatas platform, built on Apache Spark and optimized to run in Hadoop environments, leverages distributed computing, machine learning and a dynamically visual workspace that promotes transparent governance and ad hoc collaboration. Paxata data prep, powered by IntelliFusion, is designed to eliminate the need for coding, scripting and sampling. The solution is available as a service, and can be deployed in AWS virtual private clouds or within Hadoop environments at customer sites.
